Working with css background properties

Generally; CSS background properties are used to add efffects on an element background.
List of all background properties and how to use them;

background: used to set all background properties in one declaration.

background-attachment: used to set a backgroumd image as fixed or scrolls together with the rest of the page.

background-clip: used to specify the area of the backgroumd to be painted.

background-color: used to set the background color.

background-image: used to set the background image.

background-origin: used to specify where the background image(s) is/are placed.

background-position: used to set the starting position of a background image.

background-repeat: used to set how the background image will be repeated, horizontally or vertically.

background-size: used to set the background image(s) size.
